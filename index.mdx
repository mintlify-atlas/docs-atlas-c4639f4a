Composio is an SDK for TypeScript and Python that connects AI agents to external services — GitHub, Slack, Gmail, Notion, and 1,000+ more — without writing custom integrations. It handles OAuth flows, API key management, per-user sessions, real-time triggers, and MCP server generation so you can focus on building agents that get things done.

## Why Composio?

Building AI agents that interact with real-world apps requires solving auth, per-user isolation, tool discovery, and execution — before you even write agent logic. Composio solves all of this out of the box.

## Get Started

<Steps>
  <Step title="Install the SDK">
    Install the Composio core package and your framework's provider adapter.

    <CodeGroup>
    ```bash npm
    npm install @composio/core @composio/openai-agents
    ```
    ```bash pip
    pip install composio composio-openai-agents
    ```
    </CodeGroup>
  </Step>
  <Step title="Initialize Composio">
    Create a `Composio` instance with your API key and the provider for your AI framework.

    <CodeGroup>
    ```typescript TypeScript
    import { Composio } from '@composio/core';
    import { OpenAIAgentsProvider } from '@composio/openai-agents';

    const composio = new Composio({
      apiKey: process.env.COMPOSIO_API_KEY,
      provider: new OpenAIAgentsProvider(),
    });
    ```
    ```python Python
    from composio import Composio
    from composio_openai_agents import OpenAIAgentsProvider

    composio = Composio(
        api_key=os.environ["COMPOSIO_API_KEY"],
        provider=OpenAIAgentsProvider(),
    )
    ```
    </CodeGroup>
  </Step>
  <Step title="Create a session and get tools">
    Create a session for your user and retrieve provider-wrapped tools ready for your agent.

    <CodeGroup>
    ```typescript TypeScript
    const session = await composio.create('user@example.com', {
      toolkits: ['github', 'slack'],
    });

    const tools = await session.tools();
    ```
    ```python Python
    session = composio.create(
        user_id="user@example.com",
        toolkits=["github", "slack"],
    )

    tools = session.tools()
    ```
    </CodeGroup>
  </Step>
  <Step title="Run your agent">
    Pass the tools to your agent. Composio handles execution, auth, and result formatting automatically.

    <CodeGroup>
    ```typescript TypeScript
    import { Agent, run } from '@openai/agents';

    const agent = new Agent({ name: 'My Agent', tools });
    const result = await run(agent, 'Create a GitHub issue for the bug I found');
    ```
    ```python Python
    from agents import Agent, Runner

    agent = Agent(name="My Agent", tools=tools)
    result = await Runner.run(starting_agent=agent, input="Create a GitHub issue for the bug I found")
    ```
    </CodeGroup>
  </Step>
</Steps>
